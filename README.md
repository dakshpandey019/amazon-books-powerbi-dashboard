# Amazon Best Selling Books Dashboard 

An interactive and visually engaging Power BI dashboard built using Amazon Best Selling Books data. This project focuses on data visualization, reader insights, sales trends, author performance, and category analysis.


# Project Overview

This dashboard helps analyze:
- Book categories and genres
- Ratings and reviews
- Author performance
- Pricing insights
- Bestseller trends
- Publisher analysis

The dashboard is designed to provide meaningful business insights through clean and interactive visualizations.

# Tools Used

- Power BI
- DAX
- Data Cleaning
- Data Visualization


# Dashboard Features

## Executive Overview
- Total Books
- Average Rating
- Total Reviews
- Average Price
- Total Authors

## Analytical Insights
- Top Authors
- Genre Distribution
- Price vs Rating Analysis
- Publisher Performance
- Bestseller Trends

## Interactive Features
- Dynamic Table
- Slicers & Filters
- Conditional Formatting
- Interactive Charts


# Dataset

Dataset used:
Amazon Best Selling Books Dataset (500 Records)

Columns include:
- Title
- Author
- Category
- Rating
- Reviews
- Price
- Publisher
- Year Published
- Format


# Key Insights

- Fiction books dominate bestseller rankings.
- Highly reviewed books generally maintain higher ratings.
- Some low-priced books outperform premium books in popularity.
- Certain authors consistently appear in bestseller lists.
